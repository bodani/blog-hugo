#### 简介

 主要用于Restful风格接口测试

- 单一接口测试 

- 多接口组合逻辑测试

- 定时自动，生成测试报告 

- 失败通知

  测试用例断言

- 返回状态码

- 返回结果

- 超时时间

#### 基础应用需求

- Postman [测试用例编写](https://blog.csdn.net/cai_iac/article/details/81030619)

- Newman 执行测试用例命令行工具

- Jenkins 自动化集成

- Git 测试用例成果物管理

#### 目标

- 定时自动或手动触发测试,并生成测试报告

- 测试结果有异常情况通知相关人员

- 通过环境变量自适配生产或测试环境
